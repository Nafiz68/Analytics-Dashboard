# Quick Start Guide

## Getting Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Open in Browser
Navigate to [http://localhost:3000](http://localhost:3000)

## Features to Try

### Theme Toggle
- Click the moon/sun icon in the top right
- Theme preference is saved automatically

### Filters
- Change date range: Last 7 days, 30 days, or 12 months
- Filter by user type: All, Free, Premium, or Enterprise
- Watch all metrics update in real-time

### Export Data
- Click "Export CSV" button
- Downloads complete dashboard data

### Responsive Design
- Resize browser window
- Try on mobile device
- Sidebar auto-collapses on small screens

### Dark Mode
- Toggle between light and dark themes
- All components adapt automatically

## Project Structure

```
analytics-dashboard/
 app/              # Pages and layouts
 components/       # Reusable UI components
 store/           # State management
 services/        # API and data services
 hooks/           # Custom React hooks
 types/           # TypeScript definitions
```

## Available Scripts

- 
pm run dev - Start development server
- 
pm run build - Build for production
- 
pm start - Run production build
- 
pm run lint - Run ESLint

- **Next.js 15** - React framework
